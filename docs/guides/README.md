## ðŸ’¡ Key Concepts

### Network Information
- **Mainnet**: Base L2
- **Testnet**: Base Sepolia
- **Block Explorer**: [Basescan](https://basescan.org)

### Economic Parameters
- **Minimum Node Stake**: 100 ETH
- **Platform Fee**: 2.5%
- **Job Timeout**: Configurable (default 1 hour)

### Supported Models
- GPT-4, GPT-3.5
- Llama 2 (7B, 13B, 70B)
- Mistral 7B
- Stable Diffusion
- Custom models (with proof)
